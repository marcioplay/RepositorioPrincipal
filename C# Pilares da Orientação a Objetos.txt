2º Herança:
A herança, surgi para que você tenha reuso de código.

Exemplo de herança entre classes

public class Animal {
  void Respirar(){}
}

class Humano : Animal {
 // Nesse caso, Humano herda de animal.

}

Humano h = new Humano();
h.Respirar();
